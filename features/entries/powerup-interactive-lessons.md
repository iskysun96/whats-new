## What it does
The `/powerup` command launches interactive lessons that teach you Claude Code features through hands-on practice with animated demonstrations. Instead of reading documentation, you learn by doing — with Claude guiding you through each feature step-by-step while showing you exactly what's happening in real-time.

## When to use it
- When you're new to Claude Code and want to learn the core features interactively
- Before starting a complex project to brush up on advanced features you haven't used recently
- When onboarding team members who need to get productive with Claude Code quickly
- To discover features you didn't know existed through guided exploration
- When you prefer learning by doing rather than reading static documentation

## How to use it
1. Type `/powerup` in any chat to open the interactive lesson menu
2. Browse available lessons organized by feature category (Core Modes, Editor Integration, etc.)
3. Select a lesson to begin — Claude will create a dedicated learning environment
4. Follow along with the animated demos, trying each feature as it's demonstrated
5. Complete hands-on exercises that reinforce what you've learned
6. Use `/powerup progress` to see your completion status and recommended next lessons

## Pro tips
- Run `/powerup quick-start` for a curated learning path if you're completely new to Claude Code
- Use `/powerup [feature-name]` to jump directly to lessons about a specific feature you want to master
- Lessons adapt to your skill level — more advanced users get deeper technical details and edge cases
