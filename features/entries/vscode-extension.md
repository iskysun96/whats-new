## What it does
The VS Code extension brings Claude Code directly into your editor. You get inline chat, dedicated editor tabs for each session, and a sidebar where you can manage, rename, and remove sessions. It's the same powerful Claude Code experience, but without leaving VS Code.

## When to use it
- You live in VS Code and don't want to context-switch to a terminal
- You want inline code suggestions right next to the lines you're editing
- You need to juggle multiple Claude sessions for different tasks and want them organized as tabs
- You prefer a visual sidebar for managing your conversation history
- You're pair-programming with Claude and want tight editor integration

## How to use it
1. Open the Extensions panel in VS Code (Cmd+Shift+X on Mac, Ctrl+Shift+X on Windows/Linux).
2. Search for "Claude Code" and install the official extension from the marketplace.
3. Once installed, you'll see a Claude Code icon in the sidebar — click it to open the panel.
4. Start a new session from the panel or use inline chat directly in your editor.
5. Manage sessions (rename, remove, switch) from the sidebar.

## Pro tips
- Use the keyboard shortcut to quickly toggle the Claude panel without reaching for the mouse.
- You can have multiple sessions open in separate editor tabs — great for working on different features simultaneously.
- The extension respects your VS Code workspace settings, so Claude automatically knows your project context.
